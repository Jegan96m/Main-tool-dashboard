    Slackloop

    Combination tool for Slackloop analysis and splicing management.

    Functionality:

      1. Comparison of Slackloop Counts: Identifies and displays difference in the number of Slackloops between Vetro data and the source data, including detecting missing Slackloops.
      2. Duplicate Detection: Marks and highlights any duplicate parent IDs found in the Slackloop data.
      3. Length Validation: Checks and highlights any incorrect length values in the Slackloop entries.

    How to use:

    • At Original slackloop has to upload the Fonds source data as geojson format
    • At Plan slackloop has to download the slackloops data from Vetro as geojson and upload in the tool
    • The fibers F1 and F2 can also upload for reference at fiber section
    • Once done upload the tool will direct to the location
    • The legend indicates the errors for different marker colors
        • Green marker - Slackloop inserted correctly
        • Yellow marker - Incorrect Length value
        • Red marker - Slackloop not inserted in Vetro
        • Blue marker - Duplicate Slackloops parent ID
    • Click on the circles to view details

    Splicing

    Combination tool for Slackloop analysis and splicing management.

    Functionality:

      1. Splicing Details Display: Shows detailed splicing information directly for a specific handhole, eliminating the need to search through HTML files.
      2. Ease of Access: Allows producers to quickly and easily access relevant splicing data by directly referencing the specific handhole location, using the corresponding FondID

    How to use:

    • Firstly, upload the Fond splicing detail which has converted as geojson file (provided by the admin).
    • Next, upload the Handholes downloaded from Vetro as geojson file.
    • Then, upload the cables F1, F2 and Drop downloaded from Vetro as geojson file(optional).
    • When clicked on the handhole it will pop up and show the splicing details at that location.
    • Follow the splicing details to do splicing/connectivity in Vetro.
    • The legend indicates the type of handholes depend to its color.

    Address Verification

    Verifies and validates addresses directly extracted from Vetro with outsource data.

    Functionality:

      1. Address Verification: Addresses can directly validate from here using the outsource reference.

    How to use:

    1. Click choose file to upload the geojson file download from vetro.

    2. Once uploaded it will zoom to the addresses.

    • Those red circles indicate that the address and parcel ID are duplicated.

    3. When clicking on the marker, it will show the most needed details.

    • If you need to see it in Vetro, click the link, and it will open Vetro and show the attribute of the selected address.
    • Click “Next” to see feature properties.

    4. On Page 2, you can see the attribute details.

    • You can also copy the table and paste it into Excel if needed. It will copy the attributes.

    5. Once copied, paste the attribute into Excel.

    • Make sure to have the titles in row 1.

    Address Validation for biarri

    Verifies and validates addresses directly extracted from Vetro with source data.

    Functionality:

      1. Address Verification: Highlights addresses present in the original source data but not found in Vetro.
      2. Coordinate Check: Ensures that only addresses not located at the same coordinates are shown, filtering out those that match

    How to use:

    • Firstly, upload the Vetro addresses data as geojson file.
    • Next, upload the Source addresses data geojson file.
    • Then, upload the F3 cable(Drop) from the source as geojson file too.
    • It will show the addresses which is not found in Vetro and also addresses of Vetro which are not same coordinates with the source.

    Splicing Conversion

    This tool automates the splicing process and converts raw data into Vetro-friendly format.

    How to use:

    • Firstly, upload the source CSV table zipped file.
    • Next, click Processed Splicetable ZIP button, after process the button will change to Download Processed Zip, click to download.
    • Upload the the csv zipped file at Step 2 to convert as geojson.
    • Click the process CSV button, omce the process complete, the button will change to Download Geojson, click it to download as Geojson
    • The geojson file is then add to QGIS to join the attribute with handhole attribute.
